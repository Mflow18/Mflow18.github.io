The Rocky Amour Picture Show

Projet effectué lors du premier Hackaton de la Wild Code School. A l'occasion d'Halloween, nous avons élaboré un algorithme de matching qui permet de savoir, après réponse à un questionnaire savant, quel personnage du film "The Rocky Horror Picture Show" est votre âme soeur.

Lors du Hackaton, nous avons voulu mêler JavaScript et Php alors même que nous ne faisions qu'effleurer ces sujets. S'en est suivi une déception : le transfert des données récupérées en front vers la base de données n'a pas fonctionné. J'ai donc repris le projet après coup et transféré les vues sur un MVC utilisant TWIG.

Membres : Anais FRANCOIS, Alexandre BARRE, Baptiste NICOLAS, Axel REDOIS, Guillaume SKRZYNIECKI et Mathieu FLORENTIN.
